# Tangent bundle
Def. $M$ is a smooth manifold. The tangent bundle is the set $$TM:=\cup_{p\in M}T_pM$$.
Def. the bundle project is defined as $\pi:TM\rightarrow M$ by $X\mapsto p$ where $p$ is the point for which $X\in T_pM$.
